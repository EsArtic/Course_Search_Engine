This program is implement by python 3.6 with third partly packages: numpy and matplotlib.
Before running the program, please make sure that there are no version conflicts or module missing.

If you are using Linux/Mac OS, please use run.sh to run the program.
If you are using Windows OS, please use run.bat to run the program.

After running run.bat/run.sh, out output message will be redirect and write to the file "./output",
an matplotlib figure will also be displayed.

After running the program directly, the output message will be display on the screen, and the
matplotlib figure will also be displayed.
